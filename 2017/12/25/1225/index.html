gitHub的使用

一、注册gitHub账号 （新手）

1.创建个人账户
UserName: 自己的用户名（ID）
Email address:(用来认证用户)
Password: 密码
2.选择个人计划
(默认选择第一个 免费 直接下一步即可)
3.创建组织账户
组织名称（有则填 无则随意填） 必填项
Billing Email 计费邮件（可以随意填自己qq邮箱）
下一步
4.跳过 创建完成

二、新建仓库

1.点击 New repository 此时提示请确认你的邮箱地址
–> 打开的邮箱 点击链接验证
2.创建时定义Repository name（仓库名 会是你的域名下级）
选择public模式 （开源 注意私有的需要收费）

三、下载git

下载方式可选择git官网 或者盘多多等网站搜索盘内资源

四、安装git

一般而言下一步就可以 但由于功能可能以后会用到建议全装
细节提醒(
a.第一个 三选一选项时 最长一句为Use Git and optional Unix tools from the Windows Command Promtt
意思时选择此项当你用windows打开命令行工具时 Unix的所有指令也能使用
b.第二个 三选一选项 checkout window-style, commit Unix-style line endings
检查代码时用window格式 提交时用unix
c.安装完以后 cmd输入git 或者 git –version 查看是否安装成功
)

五、使用git保存代码

首先找到相应文件夹 shift+鼠标右键在此处打开命令窗口
（或者cmd进入相应目录）

1.首次使用时

-1.git init //初始化（执行后可以看到出现.git文件）
-2.git add . //将所有源代码信息存储起来 (.代表所有文件和文件夹)
-3.git commit -m “” //提交到本地 引号内部为此次提交的注释
(
此时可能被提示 Please tell me who your are 请告诉我你是谁
输入git config –global user.email ‘你的邮箱’
git config –global user.name ‘你的名字’
注：引号内部按照个人情况填写
)
-4.git remote add origin 地址 //告诉本地应该链接到的库的地址 (注https是正常链接 SSH地址是非对称加密链接 设置完公钥后无需再每次推送时输入账号密码)
-5.git push -u origin master //将本地代码推送到库中 （如果-4地址是https 此时需要输入账号密码）

2.以后的使用（更新）时

-1.git add .
-2.git commit -m “”
-3.git push -u origin master

六、使用ssh非对称加密 解决每次提交都要输入账号密码的问题

-1.使用 ssh-keygen -t rsa 命令生成公私钥 (默认生成路径为 C:\Users\wyh(管理员名).ssh )
可后加 -f “路径” 在别的位置生成
-2.复制公钥 (ssh.pub记事本打开)–> gitHub网站 个人信息settings –>
SSH and GPS keys –> new SSH key –> title自己定义 key里面粘贴公钥
